*Descripcion: El sistema trata de un blog donde los usuarios pueden leer el contenido, tambien
se puede crear usuarios y mandar mensajes entre ellos.

-------------------------------------------------------------------------------
*Estado del proyecto: En este momento el proyecto esta terminado, un cambio que se puede hacer
en el futuro es hacer que los mensajes funcionen mas como un chat que como un mail.

-------------------------------------------------------------------------------
*Guia de instalacion:
Instalar django

```shell
python -m pip install django
```

Aplicar cambios en base de datos

```shell
python manage.py makemigrations
python manage.py migrate
```

Iniciar servidor

```shell
python manage.py runserver
```

Crear usuario administrador

```shell
python manage.py createsuperuser
```
